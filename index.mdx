PhoneInfoga is one of the most advanced tools for scanning international phone numbers. It gathers basic information such as country, area, carrier, and line type, then uses multiple scanners to search for the VoIP provider, owner identity, reputation reports, and social media footprints.

## What PhoneInfoga does

<Steps>
  <Step title="Parse the phone number">
    PhoneInfoga validates and normalizes the input number into E.164, international, and local formats, identifying the country and carrier.
  </Step>
  <Step title="Run configured scanners">
    Each enabled scanner queries a different data source — Numverify, Google Search, Google CSE, OVH — and returns results concurrently.
  </Step>
  <Step title="Review the results">
    Results are printed to the terminal or returned as JSON via the REST API, ready for deeper manual analysis.
  </Step>
</Steps>
