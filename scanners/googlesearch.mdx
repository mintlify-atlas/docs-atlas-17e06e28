---
title: "Google Search scanner"
description: "Generates Google Dork search URLs for a phone number across social media, reputation sites, disposable providers, and more. No configuration required."
---

The Google Search scanner uses [Google Dorks](https://en.wikipedia.org/wiki/Google_hacking) to generate targeted search URLs for a given phone number. It covers multiple categories of sources — social media profiles, scam report databases, disposable number providers, and general documents.

<Note>
  No configuration is required for this scanner. It runs automatically on every scan.
</Note>

## How it works

This scanner **generates links only**. It does not fetch any results itself — you must open the generated URLs manually in a browser. This is a deliberate design choice to avoid [Google scraping](https://en.wikipedia.org/wiki/Search_engine_scraping) limitations that would otherwise block automated queries.

<Tip>
  If you need programmatic search results rather than manual links, use the [Google CSE scanner](/scanners/googlecse) instead.
</Tip>

## Search categories

The scanner generates dork URLs across five categories:

| Category | Sites searched |
|----------|---------------|
| **Social media** | Facebook, Twitter, LinkedIn, Instagram, VK |
| **Disposable providers** | hs3x.com, receive-sms-now.com, smslisten.com, smsnumbersonline.com, freesmscode.com, catchsms.com, smstibo.com, smsreceiving.com, getfreesmsnumber.com, sellaite.com, receive-sms-online.info, receivesmsonline.com, receive-a-sms.com, sms-receive.net, receivefreesms.com, receive-sms.com, receivetxt.com, freephonenum.com, freesmsverification.com, receive-sms-online.com, smslive.co |
| **Reputation** | whosenumber.info, findwhocallsme.com, yellowpages.ca, phonenumbers.ie, who-calledme.com, usphonesearch.net, whocalled.us, quinumero.info, uk.popularphotolook.com |
| **Individuals** | numinfo.net, sync.me, whocallsyou.de, pastebin.com, whycall.me, locatefamily.com, spytox.com |
| **General** | Broad web search and document file types (doc, docx, pdf, xls, csv, txt, ppt, and more) |

## How to run

```bash
phoneinfoga scan -n +4176418xxxx
```
